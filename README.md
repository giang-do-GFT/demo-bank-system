### Prerequisite

To run this project we need `Java 21` & `Gradle 8.7`

We can use `SDKMAN` to install them:

#### Install `SDKMAN`:

````shell
curl -s "https://get.sdkman.io" | bash
````

#### Install `Java` & `Gradle`:

````shell
sdk install java
sdk install gradle
````

### Run CLI

```shell
gradle run
```

### Run test

```shell
gradle test
```
